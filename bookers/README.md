# README


### deviseをインストール
1. Gemfileに`gem 'devise'`を追加
2. アプリに読み込ませる  
`$ bundle install`  
3. deviseの初期設定  
`$ rails g devise:install`
#### ユーザテーブル作成
`$ rails g devise User`
